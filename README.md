Simple DLNA Server in Rust:

```sh
Usage: dlnad [OPTIONS]

Options:
  -p, --port <PORT>            [default: 8200]
  -i, --ip <IP_ADDRESS>        [default: 0.0.0.0]
  -d, --directory <DIRECTORY>  [default: .]
  -n, --name <NAME>
  -v, --verbose
  -h, --help                   Print help
  -V, --version                Print version
  
Tested with MP4 files (with embedded soft subtitles) and LG WebOS client.
```
